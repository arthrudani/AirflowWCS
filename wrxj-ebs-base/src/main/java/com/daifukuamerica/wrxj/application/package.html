Application environment.

Details: This package delivers generic, application-level features, in much the same way that the {@link java.lang.Runtime} and {@link java.lang.System} classes provide runtime- and system-level features.  Thus, the main class in this package is {@link com.daifukuamerica.wrxj.application.Application}.

Currently, the only application-level feature implemented in this package is the properties stack. For more information, see {@link com.daifukuamerica.wrxj.application.PropertiesStack}.
