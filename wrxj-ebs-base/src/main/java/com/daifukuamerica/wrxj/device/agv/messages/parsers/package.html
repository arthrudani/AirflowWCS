Message parsers

This package contains classes to parse messages from the CMS. The following classes are implemented in this package:
Class Name Message Description
ACKParserImpl Class to process ACK messages.
ALMParserImpl Class to process ALM (Alarm) messages.
ENDParserImpl Class to process END (End of report) message. A report request from Warehouse Rx causes multiple responses from CMS. This message marks the end of a set of responses.
ERRParserImpl Class to handle CMS Error message. This message indicates some type of internal error on CMS and is only for notification purposes.
LALParserImpl Class to handle LAL (Load-At-Location) message. This message is received after a load is dropped-off at a location.
LPCParserImpl Class to handle LPC (Load Pickup Complete) message. This message is received after a load has been picked up.
LSSParserImpl Class to handle LSS (Link Startup Synchronization) message. This is a bidirectional message meaning Warehouse Rx initiates a synchronization request using an LSS message and CMS responds in kind with another LSS message containing its current sequence number.
MABParserImpl Class to handle MAB message. This message is received when an error occurs that prevents the completion of a move request, or if the move is manually canceled by the operator.
MRCParserImpl Class to handle MRC (Move Request Changed/Cancelled) message. This message is in response to a cancel request issued by Warehouse Rx.
NAKParserImpl Class to handle NAK (Negative Acknowledgement) message.
QMRParserImpl CLass to process QMR (Queued Move Report) message. These messages are sent in response to a request from Warehouse Rx. This message is not fully implemented at this writing.
SSRParserImpl Class to process SSR (Station Status Report) message. This message is in response to a request from Warehouse Rx for the status of all AGV stations. This message is normally sent at the beginning of the connection process with CMS.
VSRParserImpl Class to handle VSR (Vehicle Status Report) message. This message is not implemented at this writing.
