Test Toolbox tabs are configured via SysConfig.

Tab classes must extend TabbedFramePanel and have a constructor that takes a TestToolbox as a parameter.

SQL Template:


INSERT INTO SYSCONFIG (sGroup,sParameterName,sParameterValue,sDescription)
  VALUES (
    'TestToolbox:Panel',
    'TAB_NAME',
    'com.daifukuamerica.wrxj.swingui.test.toolbox.PACKAGE.CLASS',
    'TestToolbox Panel')

Samples in baseline:
